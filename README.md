Kmisc
=====

Kmisc provides a disorganised collection of small convenience and
utility functions.

Installation
------------

``` r
if (!require("devtools")) {
    install.packages("devtools")
    devtools::install_github('konradedgar/Kmisc')
} else {
    devtools::install_github('konradedgar/Kmisc')
}
```
